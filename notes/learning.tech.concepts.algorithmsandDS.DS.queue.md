•	What: First In, First Out (FIFO)
•	Operations:
•	enqueue() → add to rear
•	dequeue() → remove from front
•	Time complexity: O(1)
•	Variants: Circular queue, Deque, Priority queue
•	Use cases: Scheduling, task management, real-time data buffers
